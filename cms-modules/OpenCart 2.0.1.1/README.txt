Установка и настройка модуля Payler для Opencart v 2.0.1.1
==========================================================

В текушей версии модуль рассчитан на приём платежей только в рублях.

1. Скопировать содержимое папки "Opencart_Payler" в корень сайта.
2. В админке сайта в разделе "Дополнения/Оплата" появится модуль Payler.
   Необходимо его установить и затем зайти в настройки (кнопка "Изменить").
3. Заполняете поля на ваше усмотрение.  Для появления опции оплаты через Payler
   на вашем сайте нужно, чтобы состояние модуля было "Включен".

Адрес возврата
http://<сайт_магазина>/index.php?route=payment/payler/result&order_id={order_id}

Все вопросы можете отправлять на 24@payler.com
