Инструкция по подключения модуля оплаты Payler в UMI.CMS
========================================================

1. Скопировать содержимое папки (кроме payment.xsl) в корень сайта.

2. Запустить скрипт перейдя по ссылке вида
   http://вашдомен/add_payler_payment.php. После выполнения файл можно
   удалить.

3. В файл
   "/templates/название_темплэйта/xslt/modules/emarket/purchase/payment.xsl",
   между тэгами
   <xsl:stylesheet>
   </xsl:stylesheet>
   добавить код из "payment.xsl"

4. В файлы
   "/classes/modules/emarket/lang.php"
   "/classes/modules/emarket/lang.en.php"
   добавить элементы в соответствующие массивы из файлов lang.php и lang.en.php
   соответственно.

5. Войти в административную панель UMI.CMS. Перейдите в модуль "Шаблоны
   данных".  В списке "Способы оплаты" Вы увидите только что добавленный тип
   данных "Payler".  Зайдите в редактирование этого типа данных.

6. Добавьте группу с названием "Параметры" и идентификатором "settings".
   В данную группу добавьте два поля:
   - название "Тестирование", идентификатор "test", тип "Кнопка-флажок", также
     отметьте параметры "Видимое" и "Обязательное" для этого поля;
   - название "Ключ", идентификатор "key", тип "Строка", также
     отметьте параметры "Видимое" и "Обязательное" для этого поля.

7. Перейдите в модуль "Интернет магазин", в раздел "Оплата". Нажмите на кнопку
   "Добавить способ оплаты" и выберите "Payler". Заполните поля:
   - Название -  например, "Оплата банковской картой"
   - Тип оплаты должен быть указан "Payler"
   - Тестирование - поставьте галочку, если платежи проходят в тестовом режиме.
   - Ключ - впишите ключ, предоставленный компаний Payler.

8. В качестве адреса для перехода клиента необходимо указать:
   http://вашдомен/emarket/gateway/?order_id={order_id}
   Вместо order_id в фигурных скобках должно быть указано значение, присланное
   в Payler при создании платежной сессии.
