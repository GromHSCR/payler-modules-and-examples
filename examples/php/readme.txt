Пример выполнения оплаты из PHP
===============================

На сервер нужно установить curl для PHP:

  sudo apt-get install php5-curl
  sudo service apache2 restart

В файле settings.php указать платёжный ключ и пароль мерчанта.  Можно оставить платёжный ключ TestMerchantBM.

Открыть в браузере init_payment.php.  Будет создана платёжная сессия и на экране появится кнопка для перехода на страницу ввода карточных данных.

Ввести карточные данные.  Обращайтесь в службу поддержки Пэйлер 24@payler.com чтобы получить данные тестовых карт.

После оплаты пользователь возвращается обратно в магазин.  На тестовом аккаунте TestMerchantBM пользователь возвращается на адрес http://localhost:7820/Complete-order_id={order_id}.

Обращайтесь в службу поддержки для создания отдельного тестового аккаунта.  В настройках личного кабинета можно будет установить другой адрес возврата.

Описание API здесь http://payler.com/docs/acquiring_docs:
- Payler Gate API
- Статусы транзакции
- Коды ошибок
